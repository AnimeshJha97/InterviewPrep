# Summary

P3Kit is a Next.js app for AI resume-based interview prep.

Brand:
- Product: P3Kit
- Company: Orvion Labs
- Tagline: Prepare • Perform • Progress

Stack:
- Next.js App Router
- TypeScript
- Tailwind/global CSS
- Auth.js Google login
- MongoDB/Mongoose
- OpenAI default AI provider
- Gemini optional provider

Main areas:
- Marketing pages
- Auth
- Onboarding
- Resume upload
- AI kit generation
- Dashboard
- Progress and notes
- Activity logs
- Rate limits

- `src/app`
- `src/components`
- `src/lib`
- `src/models`
- `src/types`
- `src/data/brand.ts`
